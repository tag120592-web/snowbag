A user / entity avatar — `<TNUserPicture>`. Image, initials, or icon; optional presence dot.

```jsx
<Avatar text="ИП" status="green" />
<Avatar image="/u/12.jpg" size="lg" />
<Avatar icon="building-2" square color="var(--blue-10)" />
```

Props: `text`, `image`, `icon`, `size` (xs–xl), `square`, `status`, `color`.
