A monochrome icon that inherits text color — the TN `<TNIcon>` equivalent, backed by Lucide.

```jsx
<Icon name="search" size={20} />
<Icon name="check-circle" color="var(--content-system-positive)" />
```

Use any Lucide icon name. Tint with `color` (defaults to `currentColor`), size with `size` (px number or CSS length). Decorative by default (`aria-hidden`).
