A centered empty / zero state — `<TNEmptyContent>`.

```jsx
<EmptyContent icon="file-search" title="Нет аудитов"
  description="Загрузите документ, чтобы запустить первую проверку"
  action={<Button variant="accent" icon="plus">Новый аудит</Button>} />
```

Props: `icon`, `title`, `description`, `action`.
