# Changelog

## 6.3.2

Компонент `TNSearch`:
- Добавлен слот контента справа в поле ввода

## 6.3.0

Тёмная тема для компонентов:
- Добавлена цветопара для тёмной темы css-переменных цветов
- Добавлена стандартная палитра цветов из Дизайн-системы
- Добавлена функция переключения цветовой схемы

## 6.2.32

Компонент `TNTable`:
- Добавлен параметр `popperOptions` для всех внутренних компонентов, что используют `TNPopover`

## 6.2.32

Компонент `TNPopover`:
- Добавлено управление `z-index` в `popperOptions`

## 6.2.31

Компонент `TNPopover`:
- Добавление `teleport` в `popperOptions`

## 6.2.26

Компонент `TNMultiSelector`:
- Добавлен слот выбранного элемента и списка элементов

## 6.2.21

- Добавление постоянного отступа (`smoothMessage`) для компонентов `TNDatepicker`, `TNRangeDatepicker`, `TNInput`, `TNMultiSelect`, `TNSelect`, `TNScrollSelect`, `TNTextarea` и анимация их появления
- Добавление возможности ставить фокус на поле поиска при открытии селектора с поиском

## 6.2.19

- Компонент `TNMultiSelector`:
  - Заменен тег элемента-индикатора "стрелка" открытого/закрытого состояния селектора на `TNIcon`

## 6.2.18

- Компонент `TNTable`:
  - Изменена логика кнопки выбора всех строк таблицы (Пример: выбраны 8 из 10 строк. Поведение было: кнопка "выбрать все" только очищала выбор. Сейчас: кнопка "выбрать все" может выбрать оставшиеся 2 строки).
  - Добавлен параметр для изменения этой логики: `selectAllLogic`

## 6.2.17

- Компонент `TNSelector`:
  - Заменен тег элемента-индикатора "стрелка" открытого/закрытого состояния селектора на `TNIcon`

## 6.2.15

- Компонент `TNPopover`:
  - Добавлен `prop` `disabledWidth` (boolean) - отключает автоматический просчет ширины Popover элемента. Из-за чего шириной элемента теперь можно управлять через `css`
- Компонент `TNSearch`:
  - Добавлен `prop` `disabledPopoverWidth` - устанавливает значение для `TNPopover` внутри компонента

## 6.2.14

- Компонент `TNTable`:
  - Фикс дублирования столбцов таблицы

## 6.2.5

- Компонент `TNTable`:
  - добавлен `slot` `settings` для кастомизации окна настроек отображаемых столбцов таблицы
  - добавлен параметр `outerVisibleFields`, изменение которого меняет отображаемые столбцы
  - добавлен параметр `closeSettingsOnScroll`, отвечающий за закрытие окна настроек отображаемых столбцов таблицы при скролле таблицы
  - добавлен параметр `selectedRowId`, отвечающий за цветовое выделение строки таблицы

## 5.3.6

- Добавлен компонент `TNPopup` внутри которого используется `TNFloatingButtons`
- Добавлен компонент `TNScroll`

- Для компонента `TNSelect` добавлен параметр `scroll-selected-into-view`, скроллящий список вариантов к первому выбранному элементу
- Для компонента `TNTextarea` добавлены параметры `min-height` и `maxHeight` для управления минимальной и максимальной высотой поля ввода
- Для компонента `TNUserPicture` добавлен параметр `hide-on-error`, который скрывает картинку если произошла ошибка при её загрузке
- Добавлен composable функция `useToaster` для появления тоста без фиксации компонента `TNToaster`
- Добавлены новые иконки для `TNIcon`:

  - `lock-filled`
  - `lock-filled-small`
  - `edit-1-light`
  - `edit-1-filled`
  - `edit-1-filled-small`

## 4.0.0

### Features:

- Новый компонент `TNCard`
- Новый компонент `TNDropdown`
- Новый компонент `TNPagination`
- Новый компонент `TNMenuBar`
- Добавлены новые иконки для `TNIcon`:
  - `reload`
  - `circle`
  - `settings-2`
  - `export`
- Для компонента `TNBottomSheet` добавлен параметр`transparent-background`, отвечающий за прозрачный фон
- Для компонента `TNDatePicker` добавлен параметр `always-bottom`, при котором Дейтпикер всегда отображается снизу при десктопной ширине экрана
- Тип данных для `summary-value` и `item-value` компонента `TNRadio` изменены на строку/число
- `TNSelector`
  - Добавлен параметр `disabled`
  - Добавлена возможность добавить иконку-кнопку для элементов списка
- `TNTree`
  - Интерфейс `TNTreeProps.Option`
    - Добавлено поле `disabled`, которое отключает возможность выбирать и разворачивать элемент дерева
    - Добавлено поле `disableSelect`, которое отключает возможность выбирать, но оставляет возможность разворачивать элемент дерева
    - Добавлено поле `iconButton`, которое добавляет элементу дерева иконку-кнопку
  - Добавлен параметр `child-recursive-select`. Он отключает автоматический выбор детей элемента дерева при его выборе
  - Добавлен параметр `simple`, включающий упрощённый режим отображения элементов дерева - чекбоксы сменяются иконками-галочками для отображения выбранного элемента, а выбор происходит по большей части элемента дерева
  - Добавлен параметр `highlight-disable`, с помощью которого заблокированные элементы дерева подсвечиваются серым
- Изменён цвет css-переменной `--content-system-neutral` на `#4e5867`

## 3.0.0

### Features:

- Новый компонент `TNCell`.
- Новый компонент `TNTooltip`.
- Новый компонент `TNSearch`.
- Новый компонент `TNUserPicture`.
- Новый компонент `TNFileIcon`.
- Обновлён компонент `TNTag`
- Все scss-файлы переведены на css
- Переработаны css-переменные цветов:
  - Цвета сгруппированы на:
    - Фоновые (`--background`)
    - Текстовые (`--content`)
    - Границ (`--border`)
    - Теней (`--shadow`)
  - У каждой группы есть состояние (кроме `--shadow`):
    - `-primary` (в некоторых случаях их два: `a` и `b`)
    - `-secondary`
    - `-tertiary`
    - `-accent`
  - У каждого состояния также есть подсостояние:
    - `-enabled`
    - `-hover`
    - `-pressed`
    - `-disabled`

### Fixes:

- Названия иконок для компонентов, использующие иконки `TNIcon`, типизированы в соответствующие параметры компонентов
- `TNBottomSheet`:
  - Название блока БЭМ изменено с `bottom-sheet` на `tn-bottom-sheet`
  - Кнопка закрытия переделана на `TNButton`
  - Параметр `custom-class` может теперь принимать список строк
- `TNButton`:
  - Удалён слот `icon-left`
  - Поправлены внутренние отступы
- `TNDatepicker`
  - Для числовых значений параметров добавлена возможность передавать строки
- `TNIcon`
  - Исправлен размер иконки `favorite`
  - Добавлены иконки `order`, `certification`, `external-share`, `project` и `product`
  - Убраны параметры `width` и `height`
- `TNInput`
  - Убраны параметры `type`, `search` и `mobile-break-point`
  - Кнопка очистки поля перенесена на `TNButton`
  - Добавлен слот `icon` для отображения иконки
- `TNProgressBar`
  - Для числовых значений параметров добавлена возможность передавать строки
- `TNSelector`
  - Добавлен параметр `flat` и `empty-list-hint`
  - Кнопка очистки поля перенесена на `TNButton`
  - Поиск происходит через компонент `TNSearch`
- `TNTabs`
  - Параметр `model-value` теперь принимает строку- или число-значение, а не объект
  - Добавлено поле `secondaryText` для отображения второстепенного текста в списке элементов
